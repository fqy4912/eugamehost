## DDoS Protection: Game-Aware, Not Generic

Generic DDoS protection stops simple UDP floods. That's not what kills game servers. Game servers get taken out by A2S query floods (fake requests to game server browser protocols), connection spam targeting specific game engine vulnerabilities, and application-layer attacks that look like legitimate traffic.

EUGameHost runs 30Tbps of Anycast mitigation on the Minecraft/VPS side, with 6Tbps+ CosmicGuard coverage across other services. The filtering includes game-specific logic — for example, A2S query caching on Rust servers so they stay visible and appear online even during a query flood attack.

In practice: a customer running a Minecraft server with 128 players got hit during peak hours. The protection absorbed the attack, nobody disconnected, support responded to the ticket in under ten minutes. That outcome is the exception rather than the norm at this price point.
